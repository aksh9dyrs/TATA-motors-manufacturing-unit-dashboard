TATA Motors Limited

An AI-powered, full-stack smart manufacturing analytics dashboard combining vector search, event insights, and large language models (LLMs). It supports real-time data retrieval, similarity search using embeddings, interactive querying with LLMs, RAG evaluation, and autonomous AI agent assistance.

⚡ FastAPI
🗄️ PostgreSQL (pgvector)
🤖 Advanced AI models (Llama-4, etc.)

The dashboard is built with

🔁 Real-time machine event logging and tracking
🌐 City-wise operational monitoring
🧠 AI-driven summaries and explanations of critical events
📊 Visualization of event patterns and machine behaviors
🔍 Similarity search using vector embeddings for root-cause analysis
🤖 LLM agent assistant using Llama-4-Maverick-17B-128E-Instruct for natural language queries
📈 24/7 uptime with fallback to local inference if APIs are unavailable
